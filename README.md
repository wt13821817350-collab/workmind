# WorkMind

> Enterprise AI Workspace

WorkMind is an open-source enterprise AI workspace designed to help enterprises build AI employees through knowledge management, workflow automation, intelligent agents and tool integration.

---

## ✨ Features

- 🤖 AI Employee
- 📚 Knowledge Base (RAG)
- 🔄 Workflow Automation
- 🛠 Tool Calling
- 🧠 Agent Framework
- 👥 RBAC Permission System
- 📄 AI Document Assistant

---

## 🏗 Architecture

Frontend

- Vue3
- TypeScript
- Element Plus

Backend

- FastAPI
- SQLAlchemy
- MySQL
- Redis

AI

- LangGraph
- LangChain
- OpenAI Compatible API

Deployment

- Docker
- Nginx
